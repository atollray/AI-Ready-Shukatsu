# AI-Ready Shukatsu (就活管理)

新卒就活をローカル環境で一元管理するための Next.js アプリケーションです。
WebのSaaSに頼らず、自分自身とAIエージェントだけがアクセスできる「プライベートな就活データベース」として機能します。
このアプリを用いることで、就活においてAIの支援を効率よく最大限に受けられるようになります。

## ✨ 特徴

- **全データがローカルに保存**されるためセキュア
- **各種AIエージェントと連携しやすいように最適化**されたデータ構造
- 1カラムではなく**4つのタブ（ダッシュボード・企業管理・タスク・設定）**に整理された見やすいUI
- **PCおよびタブレット（横向き）での利用に最適化されたUI**

---

## 🚀 できること

### 1. 企業ごとの徹底した情報整理
- **ステージ管理**: 「調査中」「ES準備中」「面接中」などのフェーズを一目で把握
- **スコアリング**: 志望度・マッチ度を5段階で評価
- **情報集約**: 企業の強み、懸念点、志望理由の下書きから、採用URLまですべて一箇所に

### 2. 選考フローの一元管理
- **ES（エントリーシート）**: 設問、締切、下書き状況の管理
- **イベント・面接**: 説明会、OB/OG訪問、各面接ラウンドの日程と所感の記録
- **タスク管理**: 企業に紐づくタスク（例: 「ESの草案作成」「説明会予約」など）の進捗管理
- **日次ログ**: その日の就活の振り返りやちょっとしたメモを残す機能

### 3. AIによるアシスト連携（AI-Ready）
UIから「保存する」を押すたびに、AIが読みやすい形式でファイルが自動生成されます。
エージェントに「ESの壁打ちをして」「〇〇株式会社を追加して」と指示するだけで、面倒な入力作業や情報収集を自動化できます。

> **💡 自動追加機能 (`/add-company` ワークフロー)**
> エージェント（Satella等）に「〇〇を追加して」と指示すると：
> 1. 自動でWeb検索し、業界・本社・強みを収集
> 2. あなたのプロファイルを元に「志望動機」の下書きを生成
> 3. 自動でタスク（情報収集・説明会予約など）も作成し、ローカルに保存してくれます。

## 📂 データ構造（保存先）

データはすべて `data/` ディレクトリ配下に保存され、Gitなどでバージョン管理が可能です。
AIが全データに簡単にアクセスできるため、作業を自動化できます。

- **`workspace/meta.json`**: ワークスペースのメタ情報（version / updatedAt）
- **`workspace/profile.json`**: プロフィール情報
- **`workspace/companies.json`**: 企業データ一覧
- **`workspace/tasks.json`**: タスク一覧
- **`workspace/journal-entries.json`**: ノート一覧
- **`companies/*.md`**: 企業ごとの情報をAIが読み取れる形式でまとめたMarkdown
- **`ai/context.json`**: プロフィールに加えて、ステージ別件数・高優先企業・直近期限タスク・更新鮮度を含む意思決定向けコンテキスト
- **`ai/upcoming-events.json`**: 直近の期限や面接日程
- **`ai/notes-index.md`**: 日次ノートのタイトルと要約をAIが横断検索しやすい形式でまとめたインデックス
- **`ai/generated-company-files.json`**: 自動生成された `companies/*.md` の管理用manifest（安全なクリーンアップ用）
- **`dashboard.md`**: ダッシュボードの要約
- **`calendar.ics`**: カレンダーアプリ（Googleカレンダー等）に取り込めるICSファイル

## 🛠️ 起動・セットアップ

Node.js (v18以降推奨) と `pnpm` が必要です。

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動 (localhost:3000)
pnpm run dev
```

### 【使い方Tips】
1. 最初は「設定」タブから**自分のプロフィールと就活軸**を入力しましょう。（AIがESの下書きを考える材料になります）done
2. 企業を追加し、ESの設問や面接日程を追加していきます。
3. 変更を加えたら右上の**「保存する」**ボタンを押してディスクに書き込みます。（保存を忘れてもブラウザに一時保存されます）
